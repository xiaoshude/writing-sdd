# /writing.new

创建新文章项目。

## 用法

```
/writing.new [主题名称]
```

## 执行步骤

1. **创建目录结构**
   ```
   [主题名称]/
   ├── spec.md           # 从 templates/spec.md 复制
   ├── 素材/             # 素材收集目录
   └── README.md         # 可选
   ```

2. **初始化 spec.md**
   - 填写创建日期
   - 根据主题名称预填一句话主题
   - 设置阶段为「素材收集」

3. **交互式引导**（可选）
   如果用户没有提供足够信息，引导填写：
   - 核心论点
   - 目标读者
   - 写作模式

## 输出示例

```markdown
✅ 已创建文章项目：[主题名称]

📁 目录结构：
- spec.md（已创建，待填写核心定义）
- 素材/（素材收集目录）

📋 下一步：
1. 收集素材到 `素材/` 目录
2. 运行 `/writing.spec` 完善选题规格
```

## 注意事项

- 主题名称建议使用中文，便于识别
- 避免过长的目录名（< 30 字）
- 如果目录已存在，提示用户确认是否覆盖
