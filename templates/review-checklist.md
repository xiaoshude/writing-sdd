## Gate 3: 去 AI 味

### 高频特征词检查
- [ ] 无「本质上」（换成：说白了 / 其实 / 说到底）
- [ ] 无「十分、非常、极其」（换成：很 / 特别 / 或直接删）
- [ ] 无「揭示了、表明了」（换成：说明 / 证明）
- [ ] 无「这意味着什么？」（换成：所以呢？/ 那又怎样？）
- [ ] 无商业黑话（护城河、赋能、抓手）

### 结构检查
- [ ] 排比不超过 3 个，不过于工整
- [ ] 设问句不连续出现
- [ ] 过渡句结构不重复

### 语言检查
- [ ] 长句 < 25 字（超过则拆分）
- [ ] 技术词已口语化
- [ ] 无漏字、错字
